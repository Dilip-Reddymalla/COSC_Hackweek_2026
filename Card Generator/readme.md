**Digital Business Card Generator**

Create a polished, downloadable digital business card from a simple HTML form. Live preview updates as you type, multiple templates are included, and you can download the result as PNG or PDF.

**Files**
- **Project:** [Card_Generator.html](Card%20Generator/Card_Generator.html)
- **This README:** [readme.md](Card%20Generator/readme.md)

**Features**
- **Live Preview:** updates while you type `Name` and social handles.
- **Multiple Templates:** `template1`, `template2`, `template3` (different visual styles).
- **Download:** export generated card as **PNG** or **PDF** (uses `html2canvas` + `jsPDF`).
- **Add More Socials:** dynamic social inputs via the **Add more socials** button.
- **Responsive:** layout adapts to smaller viewports.

**Quick Start**
1. Open the generator in your browser: open the file [Card_Generator.html](Card%20Generator/Card_Generator.html).
2. Fill in your `Name` field.
3. Enter one or more social handles in the **Social Media** input(s). Use **Add more socials** to add extra fields.
4. Choose a template from the **Choose a template** dropdown.
5. Click **Generate Card** to render the card preview (required before downloading).
6. Click **Download Image** to save a PNG or **Download PDF** to save a PDF.
7. Use **Generate another card** to reset the form and create a new card.

**Implementation notes**
- The live preview container is the element with id `cardPreview`.
- Buttons and inputs of interest: `generateCard`, `downloadCard_png`, `downloadCard_pdf`, `resetForm`, `addMoreSocials`, and input ids `name` / `socials`.
- Templates are generated by the functions `template1(data)`, `template2(data)`, and `template3(data)` in the script.
- External libraries are loaded from CDN:
	- `https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js`
	- `https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js`

**Customization**
- To change styles, edit the `<style>` block in [Card_Generator.html](Card%20Generator/Card_Generator.html).
- To add or modify templates, update or add template functions (`template1`, etc.) in the `<script>` section.
- To change output filename or export options, modify the `link.download` (PNG) or `pdf.save(...)` (PDF) calls in the script.

**Troubleshooting**
- If download buttons do nothing, ensure you clicked **Generate Card** first — the page requires a generated card (`gen` flag) before exporting.
- If images look cropped or low-res in PDF, adjust `html2canvas` `scale` or the `jsPDF` page sizing in the PDF export code.
- Ensure network access for CDN scripts; if offline, download the libraries and reference them locally.
